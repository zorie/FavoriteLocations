# FavLocs Web Client

# A web client of the FavLocs server

THIS IS NEW CHANGE ---- test

## Getting started

### 1. Clone the repository
### 2. Installing:
```
    npm install
```

### 3. Start the application with the following command:
```
    npm start
```

The applications serves at *localhost:3001*



## Running the tests
```
    npm test
```
